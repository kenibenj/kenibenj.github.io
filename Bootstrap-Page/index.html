Project Outline

This site is a Character Management System (CMS) for Dungeons and Dragons characters. It maintain a database of D&D characters (including monsters and NPCs) which keep track of basic stats about each character, and users can able to create, view, edit, and delete these characters.

By default users can only view previously created characters by other registered users, but registering an account and logging in gives them access to make their own.

Learn More

Technologies Used

The backend of this site is created using the ASP.NET MVC core framework in C#. The database uses SQL server and the Entity Framework ORM.

The frontend of this site uses the Bootstrap 5 and Jquery libraries. Much of the artwork was modified/created in Adobe Photoshop.

Learn More

Current Goal

Much of the design for the layout of the frontend is complete, so the lion's share of the remaining work is getting the backend of the site developed. An incedibly basic prototype that mostly relies on Visual Studio's premade MVC webapp templates has been completed, so now the focus is on creating the real deal.

Once the backend is completed, all that is left is to deploy the code on the AWS server.
